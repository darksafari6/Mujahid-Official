# Safari Cyber Warfare Portfolio

An elite, high-performance cybersecurity portfolio featuring a horror-cyber aesthetic, 3D visuals, and interactive elements.

## 🚀 Features

- **Horror Cyber Theme:** A menacing blood-red palette with glitch effects and binary rain.
- **3D Cyber Skull:** Immersive 3D visual powered by Three.js and React Three Fiber.
- **Interactive Terminal:** Functional terminal interface for threat intelligence simulation.
- **Vulnerability Matrix:** Interactive grid for analyzing and remediating security threats.
- **Expertise Domains:**
  - Offensive Operations & Penetration Testing
  - Vulnerability Scanning & Analysis
  - Security Research & Zero-Day Discovery
  - Remediation & Security Engineering
  - AI Robotics & Chatbots
  - AI Exploitation & Scripting
- **Binary Classical Sound:** High-quality, digital-classical interaction sounds.
- **Responsive Design:** Fully optimized for all screen sizes.

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS 4
- **Animations:** Motion (formerly Framer Motion)
- **3D Rendering:** Three.js, React Three Fiber
- **Icons:** Lucide React
